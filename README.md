# Smart Task Manager

A web-based task management application built using an agentic AI workflow with AntiGravity and Claude.

## Live Demo
https://mahika7730.github.io/smart-task-manager/

## Features
- Add, complete, and delete tasks
- Pending and Completed task sections
- Total task counter
- localStorage persistence
- Responsive design

## Development Process
This project was built using four AI agents collaborating in sequence:
1. **Planning Agent** – defined features, folder structure, and roadmap
2. **Frontend Agent** – designed the responsive UI
3. **Backend Agent** – implemented task logic and localStorage
4. **Integration Agent** – merged frontend and backend, tested all user flows

## Tech Stack
HTML, CSS, JavaScript
