# SmartTask

A web-based task management application built using an agentic AI workflow with AntiGravity and Claude.

## Live Demo
https://mahika7730.github.io/smart-task-manager/

## Features

**Core**
- Add, complete, and delete tasks
- Pending and Completed task sections
- Task progress tracker
- localStorage persistence

**Task Organization**
- Priority levels (Low, Medium, High)
- Categories (Inbox, Work, Personal, Learning, Health)
- Due dates with sorting (soonest/latest)
- Filter by status (All / Pending / Completed) and category
- Sort by priority or alphabetically
- Dark / Light theme toggle

**Eisenhower Matrix**
- Visual quadrant view to prioritize tasks by urgency and importance (Do First, Schedule, Delegate, Eliminate)

**Focus Timer**
- Built-in Pomodoro timer with Focus, Short Break, and Long Break modes
- Session and focus-time tracking
- Link tasks to focus sessions

## Development Process
This project was built using four AI agents collaborating in sequence:
1. **Planning Agent** – defined features, folder structure, and roadmap
2. **Frontend Agent** – designed the responsive, glassmorphism-styled UI
3. **Backend Agent** – implemented task logic, state management, and localStorage persistence
4. **Integration Agent** – merged frontend and backend, tested all user flows end-to-end

## Tech Stack
HTML5, CSS3, Vanilla JavaScript
